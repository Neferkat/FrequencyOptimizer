FrequencyOptimizer
=======

A python package for the optimal frequencies analysis (M. T. Lam et al in prep.) of pulsars

Requires:
python 2.7 or 3.X
numpy
scipy
matplotlib


PulsarNoise
-----------

A class for describing pulsar noise parameters
